``` 
Это самое сложное упражнение до сих пор, поэтому обязательно используйте всё, что вы узнали в этой книге.
Создайте массив из 4 конных колесниц. Каждая колесница имеет имя (name) и количество лошадей (horses),от 1до 4.
Создайте компонент с именем “chariot”. Компонент “chariot” должен отображать имя колесницы и количество лошадей в ней.
Он также должен иметь кнопку действия. Текст кнопки зависит от текущей выбранной колесницы.
Более конкретно про текст кнопки:
• ‘Выбрать колесницу’ — отображается, пока пользователь не выбрал колесницу
• ‘Отклонить лошадей’ — когда у колесницы меньше лошадей, чем у выбранной колесницы
• ‘Нанять лошадей’ — когда у колесницы больше лошадей, чем у выбранной колесницы
• ‘В путь!’ — когда колесница выбрана (эта кнопка будет отключена)
Пользователь должен иметь возможность выбрать колесницу, а затем выбрать между любыми другими колесницами, которые он хочет.
Пример сценария:
Пользователь выбрал колесницу с 2 лошадьми и на кнопке надпись ‘В путь!’.
У колесницы с 3 лошадьми есть ещё одна лошадь, поэтому на кнопке текст ‘Нанять лошадей’.
У колесницы с 1 лошадью меньше лошадей, чем у колесницы пользователя, поэтому на кнопке будет ‘Отклонить лошадей’.
```
