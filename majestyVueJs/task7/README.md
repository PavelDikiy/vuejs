#Компоненты
> 7.5 Домашняя работа

``` 
Создайте массив планет.
У каждой планеты будут имя (name) и количество посещений (number of visits).
Вы можете полететь на любую планету, но ограничены только 3 посещениями на каждую планету из-за нехватки топлива.
Для решения этого задания, вам нужен компонент Planet с соответствующими методами/вычисляемыми свойствами.

Каждая планета должна отображать:
• своё имя
• количество посещений
• Кнопка Посетить (если максимальное количество посещений этой планеты ещё не было достигнуто)
• Иконка, указывающая, была ли посещена планета хотя бы один ра
```
